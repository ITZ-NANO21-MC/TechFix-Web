# **App Name**: TechFix Web

## Core Features:

- Homepage Hero Section: Dynamic hero section with a brief introduction to TechFix Solutions, key selling points, and clear call-to-action buttons that link to services and contact pages.
- Service Listings: Detailed listing of repair services offered, including clear pricing for each service. Enable easy updates to pricing and service descriptions through a CMS-like interface (consider a simple JSON-based approach for MVP).
- Interactive Photo Gallery: Showcase before-and-after repair photos and product images using a visually appealing gallery layout. Implement a lightbox feature for detailed viewing and use transitions to navigate between images. Lazy loading may be required for 20+ images.
- Accessories Catalog: A well-organized and filterable catalog of available phone and PC accessories. Include high-quality images, descriptions, and pricing. Add a 'New Arrivals' section and a search function for easy browsing.
- Contact Form with Email Integration: Implement a functional contact form for inquiries. Use EmailJS for reliable email sending without backend setup. Include fields for name, email, phone number, and message, with appropriate validation to prevent spam and ensure data accuracy.
- Floating Contact Buttons: Fixed position WhatsApp and social media buttons for immediate customer engagement. Integrate direct links to WhatsApp for instant messaging and social media platforms.
- Location Map Integration: Integrate Google Maps to display the physical location of TechFix Solutions. Ensure the map is responsive and provides clear directions or a link to Google Maps for navigation.

## Style Guidelines:

- Primary color: A technological blue (#2563eb) reflecting the tech focus of the business.
- Background color: Light desaturated blue (#E6E9F2) for a clean, modern feel.
- Accent color: A vibrant green (#10b981), complementing the primary color, for calls to action and highlights.
- Body font: 'Inter' sans-serif font for body text and headlines.
- Utilize 'React Icons' for a consistent and scalable icon set across the website.
- Implement a responsive grid layout to ensure optimal viewing experience across devices.
- Employ subtle 'Framer Motion' animations to create smooth transitions and interactions, enhancing the user experience.
